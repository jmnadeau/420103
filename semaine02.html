Pendant la séance

Introduction à HTML et à l'environnement de développement. Présentation de la structuration par des balises div

Nous ferons une première page HTML ensemble en utilisant Brackets et Chrome.

Vous devez compléter les exercices de la semaine. Vous devez également compléter la phase 1 du projet.

Exercices de la semaine

Exercice A

Le premier exercice consiste à reproduire la page qui est produite dans la video. Faite un fichier appelé simpsons.html.

Exercice B

Vous devez créer une page que vous sauverez dans un fichier premierepage.html. Vous devez mettre votre nom comme titre de la page. Son contenu doit contenir au moins 3 titres de niveaux différents (h1 à h6) ainsi qu'une liste HTML numérotée et une liste html non numérotée.

Exercice C

Créer une page que vous sauverez dans un fichier liens.html dans lequel vous incluerez des liens vers:

  • une video sur une plateforme de votre choix
  • une image disponible sur le web
  • un lien qui est en fait une image cliquable

Exercice D

Créer une page blocbloc.html. Ajouter un paragraphe qui contient un autre paragraphe contenant lui-même le texte "Bonjour". Ensuite créer un second paragraphe contenant une div contenant elle-même une image de votre choix. Finalement créer une balise span contenant un paragraphe contenant le texte "hello".

Exercice E

Rechercher ce qu'est un non-breaking space ou espace insécable. Recherchez d'autres caractères spéciaux HTML. Donnez plusieurs exemples d'utilisation de caractères spéciaux dans une page special.html. Essayez d'afficher du code HTML dans votre page HTML : 

<p>

Exercice F

Assurez-vous que toutes les pages que vous avez créées passent le validateur et sont indentées correctement.
