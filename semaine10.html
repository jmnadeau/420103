Bootstrap

Avant la séance (2h)

Bootstrap est une feuille de style permettant de faciliter la mise en page d'une page web d'apparence professionnelle rapidement. Vous devez:

  • Explorer le site web de Bootstrap ici (max 30 minutes)
  • Regarder et tenter de reproduire la video d'introduction sur Bootstrap ci-dessous
  • OPTIONNEL : regarder la 2ème video sur comment approcher la page de Facebook avec Bootstrap
Pendant la séance
Vous compléterez les exercices non couverts pendant votre travail d'avant séance.
Vous devrez également compléter la page Bootstrap du TP.
